## Run Locally

Clone the project

```bash
  git clone [https://link-of-repo](https://github.com/RAHUL956777/protfolio)
```

Go to the project directory

```bash
  cd protfolio
```

Install dependencies

```bash
  npm install or npm i
```

Start the server

```bash
  npm run dev
```
